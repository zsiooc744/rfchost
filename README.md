## 网络和硬件情况

几个关键数据直接说：

- **虚拟化**：KVM，不是 OVZ，操作系统自由度正常
- **存储**：NVMe SSD，I/O 实测均值 242~915 MB/s（不同节点差异较大）
- **处理器**：Intel Xeon 或 AMD EPYC（具体看节点）
- **IP 配置**：每台默认 1 个 IPv4 + /64 IPv6
- **流量计算方式**：部分套餐单向计（仅出站），部分双向计，下单前注意看清楚

香港 CMI 线路晚高峰速度会有所下降，但对建站和一般业务影响有限。洛杉矶 CN2 GIA 的稳定性口碑相对更好，只是货源问题比较头疼。

## 付款和使用注意事项

使用之前几个细节值得关注：

**流量用完之后**：实例直接暂停，不是限速降速，等下个计费周期才自动恢复，不支持手动重置流量。

**续费价格承诺**：官方承诺以购买价格和最新价格两者中较低的那个续费——但这个承诺不适用于促销价格购买的套餐。

**升级套餐**：需要提工单，官方会计算差价补发票，不能自助操作。

**客服时间**：仅香港法定工作日响应，每次工单只处理一个问题。整体来说响应速度还算可以，技术问题基本都能解决。
